## üöÄ Key Technologies & Concepts

### 1. JDBC (Java Database Connectivity)
*   **Focus:** Manual database connection management.
*   **Concepts:** `DriverManager`, `Connection`, `PreparedStatement`, and `ResultSet`.
*   **Outcome:** Basic console-based CRUD using raw SQL queries.

### 2. MVC Design Pattern (Model-View-Controller)
*   **Focus:** Separation of concerns.
*   **Concepts:** Servlets (Controller), JSP (View), and POJOs (Model).
*   **Outcome:** A structured web application where UI and Logic are separated.

### 3. Hibernate ORM
*   **Focus:** Object-Relational Mapping.
*   **Concepts:** `SessionFactory`, `Session`, HQL (Hibernate Query Language), and Annotations (`@Entity`, `@Table`).
*   **Outcome:** Interacting with the database using Java Objects instead of SQL.

### 4. Spring Framework & Spring MVC
*   **Focus:** Dependency Injection (DI) and Inversion of Control (IoC).
*   **Concepts:** `@Controller`, `@Service`, `@Repository`, and Bean management.
*   **Outcome:** Loosely coupled and easily testable application architecture.

### 5. Spring Boot & Spring Data JPA
*   **Focus:** Rapid development and Auto-configuration.
*   **Concepts:** `JpaRepository`, RESTful Web Services, and Embedded Servers (Tomcat).
*   **Outcome:** A production-ready CRUD application with minimal configuration.
